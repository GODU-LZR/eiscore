# 送检前终检清单

整理时间：2026-03-15
当前主稿：`docs/毕业论文总稿_按校模板终稿_修订版7_2026-03-15.docx`

## 1. 主稿确认

送检前先确认以下两点：

1. 送检使用的最终文件为 `修订版7`，不要再误用 `修订版5` 或更早版本。
2. 若后续还要补截图、改参考文献或调格式，应始终在同一份主稿上继续修改，避免多版本并行造成内容不一致。

## 2. 目录与页码

需要逐项检查：

1. Word 自动目录是否已更新到最新页码。
2. `设计总说明`、`Introduction`、第一章到第九章、`鸣谢`、`参考文献` 是否都进入目录。
3. 目录中的页码是否与正文实际起始页一致。
4. 章节标题中是否残留草稿日期、版本号或“初稿”字样。

## 3. 章节结构

需要确认：

1. 章节顺序完整，未出现重复章名。
2. 各级标题编号连续，例如 `1.1`、`1.2`、`2.1`、`2.2`。
3. `3.4.1` 至 `3.4.5`、`5.9.1` 至 `5.9.10`、`6.2` 至 `6.6`、`8.5.1` 至 `8.5.5` 等编号连续。
4. 第七章截图插图标题与正文小节一致。

## 4. 图号检查

当前论文中必须重点核对以下图号：

1. `图3-1 南派食品现有产销协同业务流程图`
2. `图3-2 EISCore核心业务用例图`
3. `图3-3 EISCore质量追溯与经营管理用例图`
4. `图4-1 EISCore系统总体架构图`
5. `图4-2 EISCore功能模块划分图`
6. `图5-1 EISCore核心数据实体关系图`
7. 第七章系统截图序列

检查标准：

1. 每张图在正文中都先有文字说明，再出现图片。
2. 图题格式统一，例如 `图4-1 EISCore系统总体架构图`。
3. 图题位置统一，通常放在图片下方。
4. 正文中的“见图X-X”与实际图号一致。
5. 不要把工作汇报图、聊天截图或不适合论文的原始调研图直接塞进正文。

## 5. 表号检查

重点核对：

1. `表3-1` 业务角色与职责表
2. 第五章物理表结构相关表格
3. 第八章黑盒测试用例表

检查标准：

1. 每张表都有表号和表题。
2. 表题格式统一，例如 `表5-1 用户表结构设计`。
3. 正文中应先说明表格作用，再引用表号。
4. 不要在正文和表格中重复解释完全相同的内容。

## 6. 参考文献

送检前必须重点检查：

1. 参考文献是否统一为 `GB/T 7714-2015` 格式。
2. 中文文献不少于 `10` 篇，英文文献不少于 `2` 篇。
3. 文末文献编号与正文引用编号一致。
4. 文献条目中是否仍残留 DOI 链接、检索网页碎片或非正式说明文字。
5. CNKI、Crossref、出版社页面导出的题录是否已经统一格式。

额外建议：

1. 最终版文献尽量以学校图书馆或 CNKI 导出的题录为准。
2. 不要保留“首批检索结果”“第二批检索结果”这类工作底稿格式到最终论文末尾。

## 7. 摘要与关键词

需要核对：

1. 中文设计总说明与英文 `Introduction` 内容是否对应。
2. 英文关键词首字母大小写、分号和换行格式是否统一。
3. 中文关键词与英文关键词是否一一对应。
4. 英文标题是否完整、无断行异常。

说明：

1. 当前本地启发式扫描剩余风险主要集中在英文摘要和关键词拼接区域。
2. 若学校系统对英文摘要敏感，不建议再大改中文正文，而应优先检查英文摘要格式和行文自然度。

## 8. 截图与实现章节

第七章是送检前最容易出问题的部分，需要确认：

1. 每张截图清晰可辨，缩放后仍能看清页面主体内容。
2. 截图与对应小节内容一致，例如员工档案页不要放到库存台账小节。
3. 页面截图不要保留浏览器无关杂项，如大量开发工具、个人聊天、系统通知。
4. 若同一页面信息过密，可裁剪核心区域，不必保留整屏无效空白。
5. 截图题目采用论文口径，不直接使用系统文件名。

建议优先补齐的截图包括：

1. 基座首页
2. 员工档案
3. 组织结构
4. 物料管理
5. 仓库管理
6. 库存台账
7. 入库页面
8. 出库页面
9. 应用中心首页
10. 流程设计器
11. 审批中心
12. 本体关系工作台
13. 移动端首页
14. 移动端盘点

## 9. 事实边界

送检前必须再核一次论文口径：

1. 已完成的模块可以写为“已完成设计与实现”。
2. 阶段性能力可以写为“已完成阶段性实现”“已完成原型验证”。
3. 不要把销售、采购、生产、质量、设备、能耗等扩展方向写成“已完整实现”。
4. AI 辅助开发部分仍应写成原型探索或阶段性验证，不写成成熟稳定平台。

## 10. 格式细节

常见终稿问题包括：

1. 标题前后空行不统一。
2. 正文段前段后间距不一致。
3. 图表前后空白过大。
4. 英文半角符号和中文全角符号混用。
5. 页眉页脚、页码格式与学校模板不一致。
6. 文中仍残留 Markdown 痕迹、链接痕迹或文件名痕迹。

## 11. 最后一次自检顺序

建议按以下顺序完成终检：

1. 锁定主稿文件。
2. 更新目录。
3. 核对章节号。
4. 核对图号和表号。
5. 核对参考文献格式。
6. 检查摘要、关键词和英文部分。
7. 检查第七章截图。
8. 从头到尾通读一遍，重点看小结、导读和展望是否自然。

## 12. 当前建议

当前阶段不建议再大幅重写正文。
更高价值的工作是完成以下收尾：

1. 统一参考文献格式。
2. 补齐第七章截图。
3. 更新目录和页码。
4. 做一次纯格式终检。
