## 4. 模拟 AIGC 检测结论

参考之前本地模拟检测结论的口径，本轮更合理的看法如下：

### 4.1 以“平均风险值”理解

当前最佳版本即“润色版”的整体本地模拟风险约为：

`9.4%`

### 4.2 以“可疑段比例”理解

当前最佳版本中：

1. 高风险 + 中风险段占比约为 `3 / 59 = 5.1%`

### 4.3 如何解释这个结果

1. 本轮新增内容经过手册约束处理后，整体风险是下降的。
2. 从基线版到润色版，平均风险分从 `0.1038` 降到 `0.0938`，中风险段从 `3` 降到 `1`。
3. 这说明新增内容经过“删虚留实、补工程锚点、减少套话”后，整体更稳。

## 5. 当前剩余高风险的主要来源

在“润色版”中，剩余的两个高风险来源主要是：

1. 中文关键词、英文 `Introduction` 与相邻标题被本地扫描器拼成同一长段
2. 轻量化语义进入 Agent 运行时的那一段，对本地模型来说仍然偏敏感

需要说明：

1. 第一类更偏向扫描器分段方式带来的误差，并不直接说明中文正文主干存在明显问题。
2. 第二类属于新加内容中最技术化的一段，本地模型对“Agent + semantic tool + tool_id”这类表达比较敏感。
3. 实际上，更激进地把它改得更具体，并没有进一步降低风险，反而略有上升。
