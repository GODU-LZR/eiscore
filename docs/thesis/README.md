# EISCore 毕业设计说明书公开存证版

本目录保存 EISCore 项目毕业设计说明书的公开存证版本，用于展示项目设计来源、系统架构、核心实现思路和作者开发时间线。

公开版文件由终版毕业设计说明书复制生成，仅新增公开存证说明页，并对可能涉及个人隐私、账号密码、服务器信息、企业内部敏感数据、批注修订和文档元数据进行最小必要处理。论文标题、章节结构、技术内容和参考文献未重写。

## 文件

- `毕业设计说明书-公开存证版.docx`：公开存证版 Word 文件。
- `毕业设计说明书-公开存证版.pdf`：PDF 公开存证版。当前环境未检测到稳定可用的 Word/LibreOffice 转换工具，需人工导出后补入。
