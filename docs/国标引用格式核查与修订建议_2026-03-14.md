题目：面向湛江中小制造企业的企业信息化系统核心设计与实现——以广东南派食品有限公司为例
整理日期：2026-03-14
用途：核查当前论文初稿中的引用方式是否符合国家标准，并给出后续修订建议。

## 一、结论

结论先说清楚：

1. 你当前正文中使用的 `[1]`、`[1-3]` 这种编号引用方式，符合 `GB/T 7714` 常用的“顺序编码制”思路，可以继续沿用。
2. 但我前面给你整理的 `真实参考文献首批/第二批检索结果` 和 `论文最终文献池`，本质上还是“写作用文献池”和“核验清单”，还不是已经完全收口的最终版国标参考文献表。
3. 也就是说，当前状态适合写初稿正文，但在交正式版本前，参考文献表还需要再按 `GB/T 7714-2015` 统一修一轮。

## 二、联网核验后的标准依据

截至 `2026-03-14`，需要区分两个时间点：

1. `GB/T 7714-2015` 目前仍然是已经长期执行的参考文献著录国家标准，很多高校和期刊当前仍按这一版执行。
2. `GB/T 7714-2025` 已于 `2025-12-02` 发布，但根据国家数字标准馆信息，将于 `2026-07-01` 实施，届时全面替代 `GB/T 7714-2015`。

结合你现在的毕业论文时间安排看，你的初稿、查重和定稿都集中在 `2026年3月至6月`，也就是 `2026-07-01` 之前。因此，如果学院没有单独发新通知，你现在按 `GB/T 7714-2015` 来整理最稳妥。

本次判断主要依据以下来源：

1. 国家数字标准馆页面说明：`GB/T 7714-2025` 已发布，实施日期为 `2026-07-01`。
来源：https://www.ndls.org.cn/resources/info/2001648569829879809
2. 对外经济贸易大学图书馆关于参考文献著录的说明，明确中文期刊文献著录参照 `GB/T 7714-2015`，并给出了期刊、图书、学位论文、论文集、网络文献的示例格式。
来源：https://lib.uibe.edu.cn/fw/xkfw/kyxjt/c7f0f363d8b546ce9ad94e4e09f85872.htm
3. 西安邮电大学关于执行 `GB/T 7714-2015` 的通知，说明该标准自 `2015-12-01` 起执行。
来源：https://xupt.edu.cn/info/2104/50729.htm
4. 海南大学学报编辑部下载中心页面，也明确挂载的是 `国家标准GB/T 7714—2015`。
来源：https://xb.hainanu.edu.cn/info/1023/1134.htm

## 三、你现在哪些地方是“基本符合”的

### 1. 正文引用方式

你现在第一章初稿里的正文引用方式，例如：

`[1]`
`[1-3]`
`[11-13]`

这种写法本质上属于顺序编码制。按照高校常见论文写法，这是可以接受的，也是目前最适合你本科论文的方式。后面整篇论文继续保持统一即可。

### 2. 文献类型标识

你当前整理出来的大部分文献已经区分了常见文献类型，例如：

1. 期刊论文用 `[J]`
2. 会议论文用 `[C]`
3. 标准用 `[S]`
4. 网络文献常见写法为 `[EB/OL]`

这一点和 `GB/T 7714-2015` 的要求是一致的。

## 四、你现在哪些地方“还不能算最终国标格式”

### 1. 当前文档里很多内容是“检索说明”，不是正式参考文献

例如前面几份文档中包含：

1. `用途建议`
2. `核验来源`
3. `核验链接`
4. `当前建议`

这些内容对于你写论文很有用，但它们不属于最终论文文末参考文献表的组成部分。正式参考文献表里不能保留这些说明项。

### 2. 有些条目还没有完全按国标字段写满

例如期刊论文的常见标准格式通常应写成：

`作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.`

图书一般写成：

`作者. 书名[M]. 出版地: 出版社, 出版年.`

会议论文常见写法一般要更完整，例如：

`作者. 题名[C]//论文集责任者. 论文集名. 出版地: 出版者, 出版年: 起止页码.`

而你现在的部分英文会议文献，比如低代码综述那篇 `[17]`，目前更偏向数据库导出的简化格式，后面还要再统一成标准格式。

### 3. 部分中文条目还需要最终从 CNKI 或学校图书馆再导出一次

原因不是这些文献是假的，而是有些站点只方便核题名和来源，不一定最适合直接作为最终版题录。尤其是：

1. 维普或网页展示页可能登录受限
2. 个别页面更像文章发布页，不是标准题录导出页
3. 同一篇文献在不同数据库中的标点样式可能略有差别

因此，当前文献池适合支撑你写作，但真正交稿前，最好再从 `CNKI`、学校图书馆或 `NoteExpress` 里统一导出一次。

### 4. 目前还没有生成“整篇论文最终参考文献表”

你现在已经有：

1. 文献检索结果
2. 最终文献池
3. 第一章中按顺序编码制插入的正文引用

但还没有一份真正按最终编号排列、能够直接放到论文最后一章前面的“参考文献”列表。这一步还需要我后面专门帮你收一次。
